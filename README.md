# Premier-League-2015-2016-Dashboard
Premier League 2015-2016 Team Stats including Goals For and against, Shots for and against, Shots on target for and against, Count of Matches by Referee and Win,Loss,Draw Percentages on a Dashboard using Microsoft Power BI

The Original Data Set `epl_2016.csv` only Contains Each Match with its stats during the season. It had been transformed to be suitable for the dashboard
